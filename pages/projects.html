Multi-User Cross-Platform Gardening App (Unity Game Engine, C#)
App created for the Human-Computer Interaction Lab at the University of Maryland, College Park. The app is created in the Unity Game Engine. 3D assets are imported from the unity store, as well as oculus integration. Custom C# scripts control player movement. Photon networking and C# scripts handle in-game networking for different objects. Custom models made in blender for the character. The app allows the user with the VR headset to interact with his/her environment. The PC users can view the environment and move around, but cannot interact with it.

Snake (Java)
Custom game of snake created using Java. Handler class obtains all game objects(player, food, player's trail) and adds them to the current game. The handler then renders the player model to the custom window created using Java's Canvas library.
